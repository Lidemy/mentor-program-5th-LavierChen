## 教你朋友 CLI

### CLI 與 GUI

當今社會，我們生活在充滿 GUI（Graphical User Interface）的世界裡，例如銀行提款機的介面，手機應用程式的點擊按鍵，都屬於視覺化的操作。而 GUI 所能實踐的功能，CLI（Command Line Interface）也都可以實踐，差別在於 CLI 屬於純文字撰寫指令與電腦溝通。

### CLI 操作教學

1. 確認目前所在位置  
   `pwd`

2. 新增資料夾 wifi  
   `mkdir wifi`

3. 切換位置到 wifi 資料夾  
   `cd wifi`

4. 新增檔案 afu.js  
   `touch afu.js`

5. 列出當前位置的所有檔案，確認 afu.js 是否新增成功  
   `ls`
